# triggers.md

> 原文档是图片版表格,此处转录为文字。事件 → 动作 总表。

| 类型 | 触发事件 | 动作 |
|------|---------|------|
| 对话 | 用户发送消息 | 实时 Claude 回复 |
| 对话 | 对话结束后 | AI 判断是否 save_memory |
| 启动 | 开机/唤醒 App | Mochi 打招呼(调用 API):冷启动 / 每日首次 / 同日再次,三态见 conversation.md |
| 状态 | 超过 2 小时无互动 | 切换「安静」动画 |
| 状态 | 超过 1 天无互动 | 切换「想你」动画 |
| 卡片 | 用户点击聊天框 ✦ | 检索当天对话 → 生成日卡 → (存档并冒泡) |
| 卡片 | 每日卡片生成完成 | Mochi 冒泡问是否深聊 |
| 记忆 | 每周定时批量 | 中期记忆压缩 |
