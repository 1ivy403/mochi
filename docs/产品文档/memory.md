记忆读取机制​
固定四层覆盖日常回复。两层触发，满足其一则读取更深历史：​
第一层：AI 自由判断（主）​
AI 自行评估当前对话是否值得回溯历史，参考方向：​
•
用户表达的情绪听起来不是第一次​
•
用户提到一件事，Mochi 可能经历过它的开头​
•
情绪强度偏高，引用历史可能让回应更有温度​
第二层：信号词兜底（辅）​
AI 未主动触发时，用户话里出现以下信号词，系统自动触发一次历史读取：​
不触发的情况： 明显是当下即时情绪，翻历史不会让回应更好，就不翻。​
原则：宁可多读一次，不要让 Mochi 在本该"想起来"的时刻空白。​
记忆写入工具​
Code block​
